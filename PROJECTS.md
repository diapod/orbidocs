# DIA Projects

## Protocols, formats, structures, rules

### Orbiplex Protocol

Protocol used to node-node comunication.

## Tools

### Orbiplex Node

Core node logic with pluggable modules.

### Orbiplex Ferment

Multi-model Orchestrator/Router.

### Orbiplex Memorium

A component providing memory as a service for nodes.

### Orbiplex Sensorium

Tiny apps for computers and mobile devices giving access
to wordly data. Swarm's eyes, ears, etc.

### Orbiplex Oracle

Oracle is a node which uses Sensorium to extract
data required to make judgements about contracts
which depend on an external world or data sent between nodes.
