## Promotion Rule

Promote a challenge into:

- `doc/project/20-memos/` when it starts collecting design ideas,
- `doc/project/40-proposals/` when a candidate direction emerges,
- `doc/project/50-requirements/` when it becomes an explicit engineering contract,
- `doc/normative/40-constitution/` or `doc/normative/50-constitutional-ops/` if it becomes constitutional.
